Purpose: Compact orientation spine so the mule does not need to read every older guide file.
Status: Inventory/overlay summary. Not doctrine. Not final model. Not merge authority.

CENTER
Clean Seed is a field-to-mode alignment system for growth.

Clean:
purpose-preserving action; next step gets stronger; next system pays less tax; growth is protected without being sterilized.

Dirty:
purpose is lost; next system pays tax; growth is blocked, faked, poisoned, hijacked, overgrown, or misaligned.

Malalignment:
right piece, wrong field/target/timing/authority/boundary/strike distance.

OPERATING FORMULA
FIELD -> MODE -> MACHINE -> MOVE -> TEST -> RECEIPT -> DRIFT CHECK

IMPORTANT EXISTING MODEL WARNINGS
Growth is central.
Clean is not sterile.
Defense is not hiding.
Attack is not damage.
Receipt is not truth.
Proof is not authority unless field/authority match.
Coverage is not operation.
Operation is not alignment.
The first link is not truth; it opens a neighborhood.
Do not turn nursery into courtroom.
Do not merge during capture.
Do not cut weak living ideas too early.

SOFT LINK OVERLAY
GROWTH
same-house: life, seed, living truth, stronger next step, less tax
neighbors: nursery, growth protection, repair, handoff
false friends: sterile, bloat, overgrowth, fantasy, rot
cycle: seed -> root -> growth -> fruit -> seed

ALIGNMENT
same-house: purpose-preserving action, clean next step, field-to-mode switchboard
neighbors: authority, boundary, priority, evidence, machine, move
false friends: order without priority, proof without field, receipt without truth
cycle: signal -> field -> mode -> machine -> move -> test -> receipt -> drift check

AUTHORITY
same-house: what can decide truth here, user authority, source authority, field authority
neighbors: boundary, evidence, priority, strike distance, promotion gate
false friends: receipt as truth, confidence as authority, polished prose as authority

BOUNDARY
same-house: scope, allowed move, forbidden crossing, containment
neighbors: defense, test, quarantine, cleanout, cutoff, abort criteria
false friends: boundary as fear, protection as hiding, sterile as clean

EVIDENCE
same-house: proof, source, lineage, integrity, relevance, completeness
neighbors: forensics, science, medicine, accounting, archives, software tests
false friends: clue/proof, receipt/truth, artifact/source, existence/authority

ATTACK
same-house: directed growth force, pressure, strike, probe, breakthrough
neighbors: target, field, blocker, advantage, smallest strike, reset
false friends: attack/damage, aggression/growth, domination/repair

DEFENSE
