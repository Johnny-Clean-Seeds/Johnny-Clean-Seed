ACTIVE QUESTION
How should the showdown engine run the fighters in the right order while preserving outside ideas, protecting ugly future value, and still letting the Judge decide by proof?

ANSWER
Use an ordered showdown.

Correct order:
1. Central Parking maps and preserves the field.
2. Weakest-Link names the active bottleneck.
3. Joker / Wildcard introduces the outside challenger.
4. STING attacks every visible claim, including Joker.
5. Shadow Victor Gate checks whether truth is still outside the whole ring.
6. Judge uses accountability ledger and rules PASS or no PASS.

ORDER LAW
The order matters because:
- Central Parking must map the field before bottleneck judgment.
- Weakest-Link must name the constraint before Joker can challenge or widen it.
- Joker must enter before STING so Joker does not escape attack.
- STING must attack before the Judge so false certainty does not reach verdict untouched.
- Shadow Victor must run before verdict so the Judge can reject the whole visible field.
- Judge must rule last with papers, proof, disproof, and no-middle PASS.

JUDGE
Role:
Allows all fighters to answer, then rules by proof.

Judge layers:
- Accountability Ledger
- Ugly Duckling Gate
- No-Middle PASS Rule
- proof/disproof checklist
- scope and boundary check
- final verdict

NO REFEREE
No Referee is added.
The Judge already ensures accountability.
A Referee would be another process object and may block progress.

JUDGE ACCOUNTABILITY LEDGER
Before verdict, the Judge records:
- active question
- scope
- fighter order used
- each fighter claim
- each fighter evidence
- each fighter risk
- each fighter disproof condition
- what happens if we do nothing
- what looks ugly but may grow
- what is false
- what is partial
- what is unknown
- what earns PASS, if anything

NO-MIDDLE PASS RULE
There is no middle PASS.

Possible statuses:
- PASS
- FAIL
- PARTIAL
- YIELD
- BLOCKED
- PARKED
- UGLY DUCKLING
- UNKNOWN

Only PASS gets crown/trust/use as proven.
PARTIAL can survive.
PARTIAL cannot wear the crown.

UGLY DUCKLING GATE
What looks bad now is not always bad later.

The Judge must separate:
- false
- unready
- ugly but possibly valuable
- true enough to pass

Ugly Duckling status means:
- not proven
- not trusted
- not installed
- not crowned
- preserved with a future test condition

The Judge does not crown ugly.
The Judge does not kill ugly by appearance.
The Judge parks it until proof shows what it becomes.

ORDER FAILURE CONDITIONS
The showdown is invalid if:
- Joker enters after STING and escapes attack.
- STING attacks before the claims are visible.
- Weakest-Link names bottleneck before the field is mapped.
- Central Parking decides truth instead of preserving the field.
- Shadow Victor becomes endless delay.
- Judge gives PARTIAL a PASS crown.
- Judge rules without proof/disproof papers.
