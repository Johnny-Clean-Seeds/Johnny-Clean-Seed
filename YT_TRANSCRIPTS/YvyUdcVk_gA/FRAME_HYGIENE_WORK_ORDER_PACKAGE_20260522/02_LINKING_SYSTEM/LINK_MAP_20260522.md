# Link Map

## Source Chain

Original subtitle source
-> clean transcript
-> transcript chunks
-> transcript receipt
-> grouped dissection
-> Clean Seed reframe
-> translation matrix
-> use ladder
-> parked next action

## Folder Roles

00_ORIGINALS:
Holds transcript evidence and source/intake material. Do not edit these as interpretation.

01_NEWER_FRAMED_WORK:
Holds our newer Clean Seed-framed work. This is interpretation/candidate material.

02_LINKING_SYSTEM:
Shows how source, interpretation, and next use connect.

03_SCAFFOLDS_AND_LADDERS:
Shows read order and use order.

04_RECEIPTS:
Proof of this local package build.

05_NEXT_ACTIONS_AND_PARKING:
Contains next safe move and parked status.

06_PACKAGE_ARCHIVE:
Holds zipped local archive after build.

## Rule

Originals must remain separated from newer framed work.
Newer framed work must point back to source but must not overwrite source.
