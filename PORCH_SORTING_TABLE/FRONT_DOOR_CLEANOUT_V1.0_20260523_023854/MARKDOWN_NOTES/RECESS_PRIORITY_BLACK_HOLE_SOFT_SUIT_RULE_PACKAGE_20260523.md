## 4. HPDR Master Algorithm

Short form:

`H P D R`

Meaning:

- Harvest;
- Prioritize;
- Digest;
- Return.

A clean RECESS pass needs:

`H=1 P=1 D=1 R=1`

If `H=0`, the answer is under-explored.  
If `P=0`, parking is passive.  
If `D=0`, bloat remains dirty.  
If `R=0`, objects are homeless.

- M = mode locked;
- H = harvested;
- B = broadened;

- K = parked overflow with return triggers;
- D = digested;
