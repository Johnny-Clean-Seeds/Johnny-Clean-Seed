## Clean Seed rule candidates

- Watts may loosen a map, but he cannot replace a receipt.
- Non-forcing means intelligent timing, not abandonment.
- If a Watts idea removes responsibility, it has been misused.
- If a Watts clip has no source trail, treat it as untrusted.
- If a Watts metaphor feels final, run map-territory check.
- If Watts becomes an authority figure, run anti-guru check.
- If Watts makes discipline feel fake, run neighbor check.

## Misuse blockers

- Block “all is one” arguments that erase consent or boundaries.
- Block “life is play” arguments that erase commitments.
- Block “ego is illusion” arguments that erase accountability.
- Block “non-forcing” arguments that excuse inaction.
- Block AI-generated Watts speech unless clearly labeled as non-Watts synthetic derivative and not used as source.
- Block unsourced quote cards.
- Block addiction romanticization.
