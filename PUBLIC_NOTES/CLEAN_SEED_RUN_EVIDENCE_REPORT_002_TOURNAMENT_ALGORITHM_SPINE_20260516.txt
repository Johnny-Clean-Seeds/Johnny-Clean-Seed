ROUTE 002A — STRUCTURE-FIRST TOURNAMENT
Theory:
Clean order comes from structure first.

Sequence tested:
Candidate Field ->
Candidate Carding Gate ->
Master Rule Interlock ->
Dropout Classifier ->
Priority Card ->
Active Core ->
Capstones ->
Lots ->
Unplaced Value ->
Cleanup Decision Map Gate ->
Public / Worker Lots ->
Bridges ->
Transition Gates ->
Report.

Result:
Strongest structural discipline.

Strengths:
- fog cannot enter raw
- every candidate must become a card
- master rules check the field before action
- cleanup cannot activate early
- lots cannot become junk drawers without capstones
- later rules land cleaner because candidates are carded and interlocked first

Weakness:
Too internal by itself.
It needs a visible Parking Control Board / Front Desk so the assistant and John can see the working state without holding the whole city mentally.

Verdict:
PASS — best structural spine, but needs usability layer.

HOW THIS RUN KNOWS THE BEST PICK
This run does not claim infinite perfect knowledge.

It proves best current route inside the declared field.

The route wins because:

1. It held more concepts cleanly than the competitors.
2. It protected authority before layout became active.
3. It allowed discovery without letting discovery become endless expansion.
4. It kept the board as display, not judge.
5. It blocked cleanup before proof.
6. It protected public/private separation.
7. It let stress-first and parking-view discoveries become salvage, not discarded loss.
8. It produced missing ideas and absorbed them into a stronger sequence.
9. It preserved the real project by running only in sandbox.
10. It creates a usable front surface without letting the whole parking city become active.

WHY OTHER ROUTES LOST
Stress-first lost as main order because it creates pressure before master rules guard the board.

Parking-view lost as main order because it makes the layout visible before authority is fully checked.

Board-early lost because the board can become judge if it appears before priority and active core are defined.

Discovery-first lost because discovery can expand the field endlessly unless a budget/field lock exists early.

The structure-first route won because it makes each later step land cleaner while still absorbing the best discoveries from the other routes.
