## Parked Candidate Methods

### Generic Backlog

Status: REJECTED AS MAIN METHOD

Useful part:

Reason parked:

Too loose. Can rot, create pressure, and let old work feel current.

### Kanban

Status: PARKED AS SUPPORT PATTERN

Useful part:

WIP limit, flow visibility, explicit policies.

Reason not main:

Controls movement but does not protect authority or proof truth by itself.

### RICE / WSJF / Scoring

Status: PARKED AS SUPPORT PATTERN

Useful part:

Reason not main:

Scores can launder weak proof into fake priority.

Status: PARKED AS SUPPORT PATTERN

Reason not main:

### Eisenhower / Urgency Matrix
