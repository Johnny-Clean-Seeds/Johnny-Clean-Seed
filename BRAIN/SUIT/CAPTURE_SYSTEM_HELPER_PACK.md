# Capture System Helper Pack

## Purpose

This pack stores the capture-hunt helpers that fit the Clean Seed / Clean Milk model fully enough to adopt as capture-system helpers.

These helpers are not loose chat and not direct active-file dumps.

They are suited helpers. They must still respect Soft Suit, Hard Suit, proof, indexing, and Respect Thy Neighbor before changing active files.

## Fit rule

A helper fits only if it:

- protects the core
- improves capture, routing, proof, memory, or recovery
- keeps capture separate from promotion
- supports list, group, rank, boss, proof, and parking
- can be tested without bloating active files
- can be stored with a retrieval key

## Soft Suit active candidates

Soft Suit can hold no more than 3 active bosses.

Current Soft Suit candidates from this pack:

1. CH-08 — No Storage Without Retrieval
2. CH-03 — Affinity Grouping Gate
3. CH-04 — Boss Severity Matrix

Reason:

These three create the strongest immediate capture improvement:

- captured ideas can be found again
- piles can be grouped cleanly
- bosses can be ranked by more than noise or urgency alone

## Promotion path

Raw capture or source ore
→ helper pack
→ Soft Suit if it becomes an active test candidate
→ combined-effect test with Hard Suit
→ proof
→ Hard Suit active pack or support link
→ controlled active-file dump only if needed

## Close condition

This helper pack passes only if the helpers remain findable, lane-safe, proof-aware, and separate from direct active-file edits.
