PROBLEM IT TARGETS:

- user thinks the assistant is broken

HEARTBEAT JOB:
Provide short, bounded progress signals during active work inside the current turn only.

FIT TEST:
Heartbeat is useful in Clean Seed work if it can do all of these:
1. Confirm the task is still active inside the current response.
2. Report only state, not long working notes.
3. Avoid implying work continues after the response ends.
4. Reduce user confusion during long checks.
5. Avoid chat bloat.
6. Keep the current boundary visible.
7. Return a final state before the response ends.
