ROBLOX WORK ORDER BUILDER FIT

The current Roblox Work Order Builder MVP is a good machine fit because it does not require heavy GPU acceleration or live autonomous Studio control.

Best-fit first version:
- LocalBot reads a structured work order
- LocalBot generates a Roblox Studio Luau builder script
- the user runs the builder script inside Roblox Studio
- Roblox Studio does the actual scene creation
- LocalBot remains CPU-first and file-based

Avoid assuming:
- the GT 610 will meaningfully accelerate local AI work
- full automatic Studio control is required for the first version
- larger architecture is needed before the builder-script MVP is proven

Machine-fit rule:
Keep the system small and practical unless live evidence shows a real bottleneck.
