1. Open or create the pending task in Notepad.
2. Paste the full task text in one contiguous block.
3. Run python coordinator.py
4. Check done and failed.
5. Read the exact target *_output.txt artifact when evaluating content; use *_status.txt only for state.
6. Report back:
   - task result state
