# Johnny Clean Seed — Public Repo Front Door

This public GitHub repository is a public reference house for the Clean Seed project.

It is not, by itself, the private local source of truth.

## Start here

Read in this order:

1. `CURRENT_TRUTH_INDEX.txt`
2. `ACTIVE_GUIDES/README_START_HERE.txt`
3. `ACTIVE_GUIDES/CLEAN_SEED_WRAP_GUIDE.txt`
4. `ACTIVE_GUIDES/CLEAN_SEED_ALIGNMENT_CHECK.txt`
5. `AGENTS.md` only when doing local file-lane / Codex-style work

## Authority map

- `ACTIVE_GUIDES/` is the active guide packet.
- Root guide files are public pointer/reference files unless `CURRENT_TRUTH_INDEX.txt` proves otherwise.
- `CURRENT_TRUTH_INDEX.txt` is the current orientation ledger.
- `AGENTS.md` is local file-lane guidance. It is not a fourth active guide.
- `PROOF_HISTORY/` is evidence, receipts, backups, reports, and audit material. It is not active doctrine by volume.
- `PUBLIC_NOTES/` is public-facing notes, handoffs, test packets, route material, and support cards. It is not active doctrine by default.
- `TEST_LANES/` is experimental/test scope only.
- `Shed/` is source-ore/reference material unless deliberately promoted through the model.

## Public/local boundary

`CURRENT_TRUTH_INDEX.txt` may mention private/local paths because it tracks the working house.

Public readers must treat those local paths as orientation facts, not commands for their own machine.

## Clean rule

Do not promote, delete, move, overwrite, or call PASS from this public view alone.

Use the active guides, current truth index, named evidence, and user approval path.
