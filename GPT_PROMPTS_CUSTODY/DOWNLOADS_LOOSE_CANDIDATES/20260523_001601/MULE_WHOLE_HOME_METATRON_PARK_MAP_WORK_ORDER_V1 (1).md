# MULE WHOLE-HOME METATRON PARK MAP WORK ORDER — V1.0

STATUS: LOCAL-FIRST MULE WORK ORDER / WHOLE-HOME NAVIGATION BUILD / PUBLIC-SAFE URL MAP DESIGN / NO CHAT-LOADED LOCAL TRUTH

MISSION:
Build a structured, local-first, zero-guess navigation system for the whole Clean Seed / Mr.Kleen / GPT_Prompts / project URL ecosystem.

The target is a Metatron Park Map: a system where a user, assistant, or mule can enter the project from the URL, local root, receipt, node, map, or handoff and still know where they are, what is active, what is source, what is private, what proof exists, what is blocked, and what closes the route.

NONNEGOTIABLE:
DONE IS NOT PASS.

PASS only means every route edge is closed, every active object has a home, every helper/package/source/debris item is returned to the correct lane, every proof claim has evidence, every privacy/authority boundary holds, and no one guessed local truth from chat-loaded material.
