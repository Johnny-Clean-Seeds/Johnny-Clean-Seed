PURPOSE:

SHOP STRUCTURE:
TOOL SHED = full inventory.
TOOL CRIB = checkout desk.
TOOLBOX WALL = job-specific kits.
SHADOW BOARD = visible missing-tool detector.
WORKBENCH = current active task.
CLIPBOARD = short job card.
RED TAG BIN = clutter, duplicate, risky, or unproven items.
REPAIR BENCH = failed tools needing sharpening.
CHARGING STATION = useful tools that need rest, evidence, or cooldown.
PARTS BINS = child tools under larger spines.
HOSE REEL = bridges/connectors pulled only as far as needed.
LOCKOUT TAG = stop unsafe use until cleared.
TORQUE WRENCH = minimal patch pressure.
IMPACT DRIVER = high-force repair, requires checkpoint.
BROOM / SHOP VAC = cleanup only after evidence capture.
WATCH = load/time/stall monitor.
STICKY NOTE = tiny temporary reminder; must be used, parked, or discarded before close.

DAILY CARRY:
Daily carry is not a belt full of tools.
Daily carry is reflex only:
1. readiness before stateful action
2. evidence before verdict
3. movement is not proof
4. do not carry the whole shed
5. return tools after task

CHECKOUT RULE:
A tool may leave the shed only when all are named:
- job
- reason
- trigger
- expected benefit
- risk
- return condition

RETURN RULE:
After use, return status must be one of:
- helped
- hurt
- partial
- unnecessary
- missing
- wrong tool
- needs repair
- merge
- park
- shed
- promote candidate only with evidence
