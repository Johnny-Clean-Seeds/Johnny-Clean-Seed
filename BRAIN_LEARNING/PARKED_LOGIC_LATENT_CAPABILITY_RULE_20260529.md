# Parked Logic Latent Capability Rule

Status: SUPPORT RULE / IDEA ROUTING
Authority: not doctrine; not ACTIVE_GUIDES; not CURRENT_TRUTH_INDEX

## Rule

Parked logic is not dead, but it is not live authority.

A parked rule, idea, source pattern, helper pattern, failed route, or unfinished lane becomes callable only when it has enough keys for the Front Gate to open it safely.

## Required keys for parked logic

- `KEY_ID`
- `NAME`
- `SHORT_PURPOSE`
- `SOURCE_ORIGIN`
- `AUTHORITY_STATUS`
- `LANE`
- `HELPER_FIT`
- `TRIGGER_WORDS`
- `FIT_CONDITIONS`
- `BLOCK_CONDITIONS`
- `PROOF_REQUIRED`
- `RELATED_FAILURES`
- `RELATED_RECEIPTS`
- `OPEN_WITH`
- `DO_NOT_USE_FOR`
- `REOPEN_TRIGGER`

## Boundary

Without keys, parked logic stays parked. With keys, it becomes latent capability that may be opened by packet, not broad authority.
