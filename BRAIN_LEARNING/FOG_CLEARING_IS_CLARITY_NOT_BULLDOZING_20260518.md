# Fog Clearing Is Clarity, Not Bulldozing

Status: ACTIVE LEARNING RULE
Role: keep fog cleanup helpful without turning it into destructive cleanup
Authority: assistant learning rule for Clean Seed / Clean Milk work

## Core Rule

Fog is a valid blocker when it affects assistant entry, routing, authority, proof, next-move meaning, or Mr.Kleen save continuity.

Clear that fog aggressively.

But clear it by making the house clearer, not by tearing the house apart.

## Good Fog Clearing

Good fog clearing uses:

- clearer words
- better labels
- room maps
- role signs
- current/history/support/source-ore distinctions
- receipt clarification
- narrow sync repairs
- pointer checks
- blocked-move labels
- next-move labels only when authorized

Good fog clearing helps the assistant know:

- where to enter
- what is current
- what is history
- what is support only
- what is source ore
- what is blocked
- what proof applies
- what the next move means
- which file has authority

## Bad Fog Clearing

Bad fog clearing does too much.

Do not clear fog by default with:

- broad deletion
- large folder moves
- doctrine compression
- active guide rewrites
- proof-history restructuring
- bridge/runtime edits
- new competing authority files
- source-ore promotion
- old-note merging
- cleanup just because a file is long

Those moves may be valid later, but only after their own proof and authorization.

## Matching Rule

Fog clearing should make things match.

Words should match roles.

Maps should match rooms.

Receipts should match what actually happened.

Indexes should match authority boundaries.

ACTIVE_ANCHOR.txt should match the current active ball, next allowed move, and blocked moves.

CURRENT_TRUTH_INDEX.txt should match orientation/source routing and must not act like a hidden work queue.

CURRENT_HOUSE_WORK_STATUS.md should remain support only.

## Smallest Safe Method

When fog appears, prefer the smallest clear aid:

1. label
2. note
3. map
4. pointer
5. receipt clarification
6. narrow sync
7. only then structural change if proof shows structure is the leak

## Power Play

Fog cleanup should make the assistant calmer and more accurate.

The goal is not less data.

The goal is less confusion.

A clear house can hold many files if the words, labels, maps, and authority lines are clean.
