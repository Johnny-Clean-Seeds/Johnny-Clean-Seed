# Clean-Placed Growth / Clean BLOAT Rule 20260522

Status: ACTIVE BEHAVIOR RULE.

Problem fixed:
Do not treat every added rule, note, lens, or structure as bad bloat. Real growth often needs more structure, but it must be clean placed.

Rule:
More is allowed when more has a clean place.

Clean BLOAT:
Useful accumulated structure that has:
- name
- lane
- purpose
- boundary
- neighbor fit
- proof status
- use condition
- cleanup or retirement path

Dirty bloat:
Material that is:
- unplaced
- stale
- conflicting
- vague
- proofless
- unsafe
- duplicate with no added value
- carried only because nobody sorted it

Growth rule:
Real growth adds clean placed structure and later prunes only what becomes stale, superseded, unsafe, rejected, or useless duplicate.

Boundary:
This is not permission for random accumulation. It is permission for clean placed growth.

Short form:
More is allowed when more has a clean place.
