# Clean Salvage Pack-Rat Source Ore Rule

Status: support behavior rule
Date: 2026-05-19

The user's pack-rat / dumpster-diver instinct is useful source behavior when kept clean.

Rule:
Do not trash odd, dirty-looking, ad-looking, old, weird, or badly placed material by default.

Also do not crown it too early.

Treat it first as source ore until identified.

Clean salvage means:
- pull the weird thing out,
- identify what it is,
- label the entry path,
- place it in the correct lane,
- keep it retrievable,
- review later for use, parking, Soft Suit, archive, or rejected-with-reason.

The goal is not minimalism.
The goal is not hoarding.
The goal is retrievable abundance.

Clean phrase:
Bag it, tag it, and later decide whether it is useful signal, parked material, or rejected-with-reason.

Boundary:
This rule does not make all material important. It prevents premature trashing and premature crowning.
