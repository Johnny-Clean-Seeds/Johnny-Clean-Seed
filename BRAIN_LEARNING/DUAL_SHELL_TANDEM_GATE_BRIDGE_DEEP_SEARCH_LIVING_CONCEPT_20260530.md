## Focal core

One visible operator shell and one visible gate shell work in tandem under the same session. The operator window is the session authority. When the operator window is open, the bridge may be open. When the operator window closes, the bridge must close, park, or stop. Work moves through keyed packets and receipts, not hidden watching.

## Definition

The Dual-Shell Tandem Gate Bridge is a visible paired-session bridge.

It is not a watcher. It is not a detached job. It is not a named-pipe implementation yet. It is not a helper that jumps into the user shell.

It is a session-tethered tandem runtime:

OPERATOR_SHELL <-> SESSION_BLACKBOARD <-> GATE_SHELL
