# Next Issue Clean Wash Signal Routine Rule

Status: SUPPORT MODEL / SOFT SUIT PROVEN TWICE / NOT DOCTRINE
Date: 2026-05-24

## Purpose

Use the next issue as the washer's test material.

This prevents the house from building a theory pile first and looking for proof later.

## Trigger

Use when:

- the user asks to wash an issue;
- a TODO says "next use" or "next issue";
- a new source, rough idea, helper output, rule candidate, or structure metaphor appears;
- the assistant is tempted to make a signal clean too fast;
- a prior run says "test on the next real candidate."

## Active Slots

Keep no more than three active Soft Suit slots:

1. Hold Signal Before Shape.
2. Gate Relationship Grammar Card.
3. Clean Wash Signal.

## Routine

1. Take a smoke break if changing task or repeating a wrong branch.
2. Name the next issue.
3. Hold the signal before shape.
4. Ring-label it: R5 source, R4 hold, R3 route, R2 proof, R1 candidate, R0 active core.
5. Pick object type: signal, proto-object, candidate, parent gate, checkpoint, helper, neighbor, card, ledger, parking, proof object, or promotion candidate.
6. Name one relation verb.
7. Name authority line.
8. Name proof need.
9. Name exit path.
10. Save the run summary to parking/support instead of pasting it by default.
11. Rerun the same issue.
12. Stop when two passes produce the same clean route with no new conflict, proof need, promotion pressure, or file custody action.
13. Feed the next issue.

## Clean Wash Signal

Clean Wash Signal means:

- same ring label;
- same object type;
- same relation verb;
- same authority line;
- same proof need;
- same exit path;
- no new route;
- no new conflict;
- no new promotion claim;
- no new file custody action.

Close as:

`CLEAN WASH SIGNAL / SUPPORT ONLY / NEXT ISSUE READY`

## Boundary

- Not doctrine.
- Not ACTIVE_GUIDES.
- Not CURRENT_TRUTH_INDEX.
- Not automation.
- Not broad cleanup.
- Not permission to promote.
- Not a replacement for Final Judge.

All three trials reached Clean Wash Signal without active-guide rewrite, current-truth rewrite, or promotion.
