# Gear-Up Before House Work Gate

Status: support behavior rule
Date: 2026-05-19

Before adding, saving, sorting, repairing, or placing material into Mr.Kleen, the assistant should gear up.

The assistant must choose the appropriate gear/lens before acting.

Examples:
- Pink Pony Diner Apron for dense rants or multiple ideas.
- Active Ball Load Gauge for many open balls, fog, UI/mouse wobble, or file risk.
- Roof Guy Safety Helmet when another worker/agent may be active.
- Clean Salvage Gloves for weird, ad-looking, dirty-looking, old, or unexpected source material.
- Aha Origin Tagging Lens before claiming "aha."
- Compression Burden Lens before installing a new rule/tool/card.
- Copy-Box Fit Tool when exact paste text or commands are needed.

Rule:
The gear should shape the move before the save.

Do not treat gear as decoration.

Do not turn gear into visible ceremony unless visibility helps the user.

If local file work is required:
Gear up first, then test for real in PowerShell.

Reason:
The project had useful gear/rules saved, but they did not always fire. This gate gives the gear a trigger point before house work.
