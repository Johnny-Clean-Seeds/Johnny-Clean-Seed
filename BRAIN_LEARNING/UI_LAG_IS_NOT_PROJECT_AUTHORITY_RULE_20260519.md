# UI Lag Is Not Project Authority Rule

Status: support behavior rule
Date: 2026-05-19

Browser or ChatGPT UI lag must not become authority over the Clean Seed / Mr.Kleen project shape.

Do not reduce, flatten, or weaken the project merely because the UI struggles with long responses, code blocks, scroll weight, or rendering.

Allowed adjustment:
Only change working shape when the change improves the project itself.

Valid reasons:
- safer copy/paste,
- clearer placement,
- cleaner proof,
- better file safety,
- less needless repetition,
- better artifact/object placement,
- clearer linked workflow.

Invalid reason:
- making the project shallower just because the browser lags.

Rule:
If the work genuinely requires depth, files, command blocks, proof, receipts, or connected explanation, keep the needed depth.

UI lag is a clue, not the boss.
