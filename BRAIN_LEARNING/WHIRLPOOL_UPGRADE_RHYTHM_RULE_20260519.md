# Whirlpool Upgrade Rhythm Rule

Status: support behavior rule
Date: 2026-05-19

Real progress can be felt before the working part is fully optimized.

Do not keep hammering a working thing just because it might yield more.

Overworking a working part can damage flow.

Better rhythm:
- let the working part move,
- work neighboring lanes,
- let new pressure reveal what actually needs upgrade,
- return when timing, friction, or proof shows the upgrade is needed.

This is the whirlpool:
motion,
orbit,
return,
respectful upgrade.

Boundary:
This does not block improvement. It blocks compulsive overworking.

Upgrade should come through rhythm and return, not panic or perfection pressure.
