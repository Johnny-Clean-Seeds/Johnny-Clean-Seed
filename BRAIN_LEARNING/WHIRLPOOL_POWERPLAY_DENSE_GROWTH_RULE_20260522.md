# Whirlpool Powerplay / Dense Growth Rule

Date: 20260522_035009

Status: active Whirlpool candidate rule / not broad cleanup permission.

## Rule

A Whirlpool run should not remove bloat merely because it looks bloated.

Bloat is first treated as compression material.

Capture its shape, repeated pressure, neighbor links, proof issues, lane confusion, naming overlap, and failure family.

## Powerplay definition

A powerplay is a clean higher-order move that collapses several related issues at once.

It may be a rule change, lane change, naming correction, proof gate, routing fix, lifecycle distinction, recovery pattern, or authority-boundary clarification.

It is only a powerplay if it reduces scattered mess while preserving useful source, proof, authority boundary, recovery path, and future movement.

## Dirty collapse

A collapse is dirty if it only deletes discomfort, hides source material, erases proof, weakens retrieval, damages neighbor lanes, or converts untested material into authority.

## Dense growth

Dense growth happens when the system keeps the lessons of the bloat but removes the repeated drag.

The final form should be smaller, stronger, easier to fire, easier to prove, and harder to misuse.
