
## 12. What This Sample Teaches

The helper is actually helping when it provides:

```text
one file
one run method
one output location
one close marker
one fix path
```

It is not helping when it adds:

```text
big proof chain
repo save ceremony
dashboard
broad scanner
multi-file framework
```

For this sample, the right helper mode is:

```text
minimum-helper Level 2/3
```

not:

```text
no-helper chaos
```

and not:

```text
full support-pack Level 4/5
```
