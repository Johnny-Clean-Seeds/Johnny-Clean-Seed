# Fix-and-Park Every Encountered Problem Rule

Date: 20260522_035009

Status: active behavior rule / problem-handling guard.

## Rule

When a problem appears during work, do not only patch around it and continue.

Repair the immediate route when safe, then preserve the durable lesson in the correct lane.

## Parking meaning

Parking is not promotion.

Parking preserves a useful pattern, failure family, correction, candidate, proof need, or future rule seed without giving it authority it has not earned.

## Integration standard

If the durable concept truly belongs inside the current artifact, integrate the cleaned concept at the right level.

If it does not belong inside the current artifact, park it separately.

Nothing useful gets lost. Nothing unproved gets promoted.
