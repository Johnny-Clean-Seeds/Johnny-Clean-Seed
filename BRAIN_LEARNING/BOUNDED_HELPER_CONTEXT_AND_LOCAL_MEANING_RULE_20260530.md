# Bounded Helper Context and Local Meaning Rule

Status: SUPPORT RULE / ADDENDUM
Authority: not doctrine, not ACTIVE_GUIDES, not CURRENT_TRUTH_INDEX

## Rule

Each helper operates inside a bounded context. Shared words may mean different things in different helper lanes.

A NEED-to-KNOW slice must define the helper's local meanings for PASS, FAIL, BLOCKED, WATCH, proof, and close.

## Examples

Code Gate PASS means parser/risk pass, not job PASS.

Save Helper PASS means write/commit/push/origin match/final clean proof.

Viewer Edit-Copy Helper PASS means editor closed, edit-copy state proved, and real proof file untouched.

## Required fields

- helper context name;
- local meaning of PASS;
- local meaning of FAIL;
- local meaning of BLOCKED;
- words that change meaning in this context;
- proof standard for this helper.
