# Center Point / Cube Home Geometry Rule

Status: SUPPORT RULE / SPATIAL ORDER  
Date: 2026-05-23  
Scope: task maps, house maps, recollection, recursive orientation

## Rule

The center is the point.

The cube is the home.

## Center point

The center point is:
- active boss;
- seed;
- core purpose;
- anchor;
- reason the task exists.

Orientation returns to this point.

## Cube / home

The cube is the bounded container where the work lives:
- rooms;
- faces;
- edges;
- lanes;
- walls;
- boundaries;
- addresses;
- proof surfaces.

## Method

Find the point first.

Then walk the cube.

Then return to the point and judge.

## Short form

Point gives orientation.

Cube gives containment.

Without point, recollection wanders.

Without cube, work has no home.
