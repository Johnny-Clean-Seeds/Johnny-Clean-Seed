# Weakness as Next Growth Cycle Rule

Status: SUPPORT RULE / GROWTH-CYCLE JUDGMENT  
Date: 2026-05-23  
Scope: build review, phase review, maturity judgment, next-cycle planning

## Rule

Weak often means the build has not reached the next cycle of growth yet.

Weak does not automatically mean bad, worthless, or wrong.

A build can be temporarily weak because it is:
- immature;
- unwrapped;
- unproven;
- waiting for next phase;
- missing its next cycle of growth.

## How to judge

When weakness appears, identify:
1. current cycle;
2. what is already strong enough;
3. what proof is missing;
4. what wrap is missing;
5. what next growth cycle requires;
6. whether to strengthen, park, or advance.

## Short form

Weakness is growth-stage information.

Find the next cycle.
