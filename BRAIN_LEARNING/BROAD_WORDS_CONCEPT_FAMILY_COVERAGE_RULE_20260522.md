# Broad Words / Concept-Family Coverage Rule

Date: 20260522_035009

Status: active wording and coverage rule.

## Rule

A rule should cover the concept family it is meant to guard.

It should not only catch the last literal failure.

It should not become a vague slogan.

## Clean broad wording

Clean broad wording names the pattern in a way that can travel across neighboring forms, names, routes, timings, tools, states, files, agents, appearances, and proof surfaces.

It stays broad enough to catch variants and precise enough to tell the assistant what to check, stop, route, prove, recover, save, park, or refuse.

## Narrowness failure

A rule is too narrow when the same failure can pass through a nearby door under a different name.

## Vagueness failure

A rule is too vague when it sounds correct but cannot guide action.

## Growth use

Useful elaboration is allowed when it improves detection, routing, proof, recovery, neighbor awareness, or later Whirlpool compression.

Later compression is allowed only when the compressed form preserves the coverage.
