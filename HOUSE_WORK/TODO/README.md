# TODO Room

Status: ACTIVE SUPPORT ROOM
Role: store TODO lists, raw tasks, grouped work items, and boss candidates before action
Authority: support room only; not command authority

## Purpose

This room holds TODO items so they do not stay loose in chat, receipts, status files, source ore, or memory.

A TODO is not automatically an authorized repair.

A TODO becomes action only after it is selected by current user command or ACTIVE_ANCHOR.txt.

## Room Rules

- Capture first.
- Sort before repair.
- Group related items.
- Separate parent bosses from symptoms.
- Do not scatter-fix.
- Do not treat TODOs as commands.
- Do not promote source ore just because it appears on a TODO list.
- Do not use this room to override ACTIVE_ANCHOR.txt.

## File Roles

TODO_CAPTURE_INBOX_20260518.md

Raw capture lane for new TODOs before sorting.

TODO_INDEX_20260518.md

Sorted TODO map with groups, status, priority, parent boss links, and next action.

## TODO Status Labels

Use these labels:

- RAW
- SORTED
- PARKED
- BOSS CANDIDATE
- ACTIVE BOSS
- BLOCKED
- DONE
- SUPERSEDED

## Required Fields For A TODO

Each TODO should record:

- name
- status
- source
- room/lane
- why it matters
- neighbor risk
- group
- boss/symptom/child/ghost/tool/proof need
- next action
- proof needed before save
