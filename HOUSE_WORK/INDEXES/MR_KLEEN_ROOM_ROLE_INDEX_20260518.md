# Mr.Kleen Room Role Index

Status: SUPPORT MAP
Authority: support only; not command authority

## Purpose

The exact file tree shows what exists.

This room role index explains what the major rooms mean.

This file prevents visual tree fog by giving each room a job, tier, entry trigger, command boundary, neighbors, non-inference rule, and proof path.

## Core Rule

A room existing does not mean it commands.

A file being visible does not mean it is active.

A source item being saved does not mean it is installed.

A proof receipt being present does not mean it is doctrine.

Current authority still comes from the active command chain.

## Tier Key

ACTIVE SPINE:
Files/rooms that guide entry, current orientation, active ball, and assistant operating behavior.

SUPPORT CONTROL:
Rooms that help sort, map, index, or manage work but cannot select action alone.

EVIDENCE:
Receipts and proof records. Evidence supports judgment but does not command by volume.

SOURCE / PARKED:
Raw ideas, candidates, outside material, and parked ledgers. Source material may support future work but cannot command.

TOOLS / TESTS:
Scripts, local tools, experiments, and test lanes. Use only when selected by current task and proof need.

ARCHIVE / HISTORY:
Old backups, quarantine, previous builds, and historical material. Enter only when current proof or rollback needs it.
