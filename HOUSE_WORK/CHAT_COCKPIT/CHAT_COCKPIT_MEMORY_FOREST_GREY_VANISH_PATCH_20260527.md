## Live behavior patch

Old chat should not be carried as raw live weight.

When old chat becomes heavy:

- extract live ideas;
- attach them to concept trees;
- merge duplicates;
- cross-link touching trees;
- keep proof-custody material safe;
- grey raw residue;
- vanish only from live carry when safe.

## Do not

Do not ask for old files or old chat again by default.

Do not carry full old explanations when a tree pointer is enough.

Do not vanish evidence.

Do not replace strong reasoning with weak summary.

## Reading method

current active object -> matching tree root -> branches -> cross-links -> proof pointers -> raw source only if needed.

## Short form

Tree the meaning. Grey the residue. Keep proof safe.
