# Frame-to-Script Candidate Suit Card V1

Date: 2026-05-30
Status: CHAT COCKPIT SUPPORT CARD / NOT DOCTRINE

## Wear line

Frames request candidates. Keys route frames. Ledger/map orders. Policy bounds. Helpers write artifacts. Code Gate tests. Proof and placement decide use. Nothing auto-runs.

## Do

Keep frame, candidate, and tool separate.
Require contract validation.
Require exact input/output.
Require policy gate.
Require candidate lane.
Require Code Gate route.
Require provenance/receipt.
Require placement decision.
Require join-back.

## Do not

Do not let a file make its own script.
Do not let a key become authority.
Do not treat candidate as tool.
Do not treat Code Gate PASS as job PASS.
Do not let .cmd hide power.
Do not promote from one proof scope.
Do not skip Final Judge.
