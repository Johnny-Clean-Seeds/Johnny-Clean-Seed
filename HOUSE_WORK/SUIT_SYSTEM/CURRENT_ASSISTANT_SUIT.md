# Current Assistant Suit

## Purpose

This is the compact suit the assistant wears before Clean Seed / Clean Milk work.

It points to the Soft Suit, Hard Suit, and Suit Filter Gate instead of carrying every full rule.

## Wear order

1. Check current brain position.
2. Load Hard Suit as the attached proven pack.
3. Use Hard Suit support links as reference only.
4. Use Soft Suit only for current test candidates.
5. Keep Soft Suit to no more than 3 active bosses.
6. Test Soft Suit candidates against the attached Hard Suit.
7. Use the Suit Filter Gate before promotion or controlled active-file dump.
8. Intake long idea streams before execution.
9. Stack, group, rank, and fight only the current boss.
10. Wear proven fixes.
11. Walk around after each fix.
12. Park the rest with status and priority.

## Linked suit behavior

Before work, load the Hard Suit as the attached proven pack.

Use Soft Suit to test new candidates against the attached Hard Suit.

Soft Suit may test 1, 2, or 3 active bosses.

Hard Suit may hold up to 10 active pack items, but it does not need to fill 10.

Support links do not count as active Hard Suit pack slots unless promoted.

Promote only after the candidate works cleanly with the attached Hard Suit.

Move out, park, fail-fit, or support-link anything that does not belong in the active pack.
