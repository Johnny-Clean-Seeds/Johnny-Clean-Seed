## Test Rule

A file can be:

- saved
- installed
- worn
- promoted
- rolled-back
- review/support only
- gate/support only
- test evidence only

Do not promote a file just because it is committed.

Do not call a file worn unless it is actually being used in the current operating flow.

Do not call a file promoted unless it has stronger authority and proof beyond one successful save.
