# Method Comparison Matrix

Status: support comparison only.
Authority: evidence / source-ore comparison.

## Internal Methods

| Method | Best Use | Advantage | Tradeoff | Use Now? |
| --- | --- | --- | --- | --- |
| Awareness before action | State classification before route choice | Prevents nearest-fix bias | Too broad if not triggered | Yes |
| Batch Safe Bin / Trial Suit | Connected batches, mule returns, companion files | Faster and safer than isolated item judging | Can hide risky individual files if not filtered | Yes |
| Adapt / Adopt Critique Gate | Outside reports and critiques | Prevents newest-report authority drift | Can become paperwork if no next-action impact | Yes |
| Current Work Queue | Loss prevention and re-entry | Prevents chat-only loss | Can become save reflex | Yes, only by threshold |
| Guard-Fail Stop Card | PowerShell/git/file safety | Stops false-pass after guard fail | Narrow to custody/file lanes | Yes |
| Gear Rack / Tool Shed | Tool selection | Avoids carrying every tool | Can become tool discussion | Yes, lightly |
| Provenance Thread Lens | Custody chains and handoffs | Clarifies source -> transform -> proof -> boundary | Friendly narration can over-clean weak chains | Yes, as Soft Suit only |
| 3/6/9 Sieve | Method comparison and candidate reduction | Preserves bench while reducing clutter | Too heavy for small tasks | Park for major comparisons |
| Bitstring tags | Compact retrieval checks | Helps filter and compare states | Can hide meaning if overused | Use readable tags first |

## Advantage Proof Summary

Prior-Mistake Retrieval Gate beats the current candidate when:

- The current task resembles a known failure.
- A batch relationship is visible.
- There is stale-state or custody risk.
- Prior proof can choose a safer route quickly.

The current candidate loop should remain unchanged when:

- The task is genuinely novel.
- No prior case exists.
- The route is already obvious and low risk.
- A full mule critique is specifically requested.

Decision for now:

Use the smaller retrieval gate first. Keep the 12-step loop as a background expansion model for big labs, not point-of-work daily carry.
