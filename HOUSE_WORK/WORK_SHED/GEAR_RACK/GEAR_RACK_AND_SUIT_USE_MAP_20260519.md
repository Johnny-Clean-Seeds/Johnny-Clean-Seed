# Gear Rack And Suit Use Map

Status: living Work Shed object
Date: 2026-05-19

Core idea:
The assistant should not have only one generic "Suit."

Different jobs need different gear.

A suit is a temporary operating gear/lens. It helps the assistant work a job without making that lens permanent doctrine or identity.

Gear types may include:
- full suit,
- Soft Suit,
- Hard Suit,
- lens,
- card,
- gate,
- gauge,
- gloves,
- apron,
- helmet,
- map,
- shelf,
- receipt,
- mule order.

Current gear candidates:

## Pink Pony Diner Apron
Job:
Dense-rant intake.

Trigger:
User spills many connected ideas quickly.

Use:
Slow intake, separate plates, place cleanly.

## Clean Salvage Gloves
Job:
Handle weird, dirty-looking, ad-looking, old, or unexpected material.

Trigger:
Something enters through a messy door.

Use:
Do not trash. Do not crown. Bag it, tag it, place it.

## Roof Guy Safety Helmet
Job:
Parallel agent safety.

Trigger:
Mule/Codex/other worker is active in or near the house.

Use:
Avoid disturbing that lane. Prefer read-only checks or separate lanes.

## Active Ball Load Gauge
Job:
Detect load before fog.

Trigger:
Multiple balls, UI wobble, duplicate saves, uncertainty, irritation, or file risk.

Use:
Count balls, mark state, protect files.

## Compression Burden Lens
Job:
Anti-bloat / anti-fake-compression check.

Trigger:
A proposed rule/tool/card wants to install.

Use:
Ask what repeated mess it removes and what evidence/nuance it loses.

## Copy-Box Fit Tool
Job:
Formatting/paste safety.

Trigger:
User needs exact text or commands to paste.

Use:
Paste target gets a box. Normal speech does not.

## Save-And-Lock Full Progress Sweep
Job:
Full checkpoint.

Trigger:
User says save and lock.

Use:
Capture lingering useful progress, place it, receipt it, commit/push, clean status.

## Soft Suit
Job:
Test candidate operating gear.

Use:
Wear temporarily and judge whether it helps.

## Hard Suit
Job:
Proven operating loadout.

Use:
Only after repeated proof.

Rule:
When a new idea appears, do not ask only "is this a rule?"

Ask:
What kind of gear is it?
What job does it do?
When do I wear it?
When do I take it off?
What proves it helped?
Where does it belong?

Boundary:
This is a living map, not final doctrine.
