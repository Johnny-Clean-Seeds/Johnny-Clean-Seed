# Gear-Change Closeout Working Order Map V1

Date: 2026-05-30
Status: ORDER MAP / SUPPORT

## Closeout order

1. Stop new work.
2. Name the gear change.
3. Identify current repo position.
4. List proven layers.
5. List blocked layers.
6. Burn dead chat weight.
7. Refresh suit with only live signal.
8. Re-walk the completed job.
9. Catch lost ideas / gaps / TODOs.
10. Choose next rope.
11. Save lock if the closeout rule/suit changed.
12. Proceed only after clean.

## Why-chain

Stop before cleanup:
new work hides old gaps.

Name gear change before repo state:
the context explains why the closeout fired.

Repo before proof layers:
state anchors the evidence.

Proven before blocked:
known ground first, danger/limits second.

Burn dead weight before suit refresh:
stale bulk contaminates the suit.

Suit refresh before re-walk:
the re-walk needs the correct lens.

Re-walk before next rope:
lost items must be recovered before selecting direction.

Next rope before save:
the save must preserve the correct handoff.

Save before proceed:
unlocked behavior is easy to lose.
