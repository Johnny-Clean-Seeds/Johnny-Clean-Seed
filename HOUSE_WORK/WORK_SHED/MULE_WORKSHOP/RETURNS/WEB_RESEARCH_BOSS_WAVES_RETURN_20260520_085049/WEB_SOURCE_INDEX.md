# WEB SOURCE INDEX

Purpose:
External idea fuel for the boss waves. Sources are not command authority. Each source was used for a principle, then tested against house fit, bloat risk, authority boundary, and proof path.

## Search Shape

Search themes:
- blameless postmortems and action items;
- runbooks/playbooks and operational excellence;
- configuration baselines and change control;
- Toyota Production System / kaizen / jidoka / waste;
- PDSA cycle;
- root cause and Five Whys;
- lessons learned systems;
- DRI/RACI ownership;
- OODA/Cynefin decision loops;
- ADR / decision record patterns.

## Fit Rule

External idea must pass all four local filters before it can be recommended:

1. It must fit the active boss.
2. It must reduce confusion more than it adds ceremony.
3. It must preserve authority boundaries.
4. It must have a small proof path inside the current house.

Verdict:
External research helped most where it gave small working shapes: runbook/playbook split, PDSA study loop, postmortem/AAR action tracking, configuration baseline, DRI ownership, and ADR decision records. Full frameworks are too heavy for direct import.
