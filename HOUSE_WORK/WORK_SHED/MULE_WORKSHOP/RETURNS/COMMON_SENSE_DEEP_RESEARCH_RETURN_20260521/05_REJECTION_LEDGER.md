## Main Rejected Shortcut

Rejected shortcut:
"Common sense is what normal people know."

Reason:
This hides every important question:

- normal for whom?
- know by what evidence?
- in which culture?
- in which domain?
- under what risk?
- before or after feedback?
- what would defeat it?

Replacement:
"Common sense is ordinary-world fit judgment that uses shared defaults, but each default needs a scope and a defeat condition."

## Watch Items

WATCH:
The phrase "common sense" is rhetorically dangerous. It can be used to shame, command, dismiss proof, or smuggle local assumptions into universal status.

WATCH:
The phrase is also genuinely useful. Removing it entirely would lose a real human capacity for quick fit judgment.

Clean route:
Keep the phrase, but require:

- active meaning;
- scope;
- source layer;
- risk level;
- defeat condition;
- feedback path.
