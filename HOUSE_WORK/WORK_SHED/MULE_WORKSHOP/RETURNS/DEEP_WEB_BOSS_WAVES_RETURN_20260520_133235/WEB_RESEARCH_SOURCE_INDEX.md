## Research Verdict

The best outside patterns are not full frameworks. The useful pieces are small control shapes:
- failure-mode thinking for "how could this break";
- unsafe-control-action thinking for "what control command is wrong";
- A3/gemba for "go see and grasp the actual situation";
- Kanban WIP limits for "stop output burden";
- FRACAS/postmortem for "failure to corrective action to verification";
- NIST baseline/RMF for "govern/map/measure/manage role discipline";
- Team Topologies for "reduce cognitive load by assigning support roles correctly."

External research does not change the active authority. It gives better candidate fixes to test.
