### C. Creative Room Resource Pass

Creative Room is not only “be creative.”

It checks:
- ideas
- tools

### D. Creative Room Candidate-Creation Rule

Creative Room may also generate new candidate rooms, tools, and methods when the active task exposes a missing work surface.

Allowed:
- propose a new room;
- name its job;
- define its boundary;
- define what it receives and returns;
- define what proof it needs;
- keep it candidate until accepted.

Blocked:
- installing a room because it sounds cool;
- creating many new rooms without fit;
- turning messy source ore into doctrine;
- skipping proof;
- treating a candidate room as already saved;
- using the Creative Room to bypass the actual lane.

### E. Messy End Path / Painting Layer Rule

Some work must be allowed to look rough before it gets clean.

This is not dirty bloat by default.
It is source-shaping.

Allowed messy stage:
- over-include useful material;
- combine rough ideas;
- sketch several possible room/tool shapes;
- make a painting-like first layer;
- preserve what is alive before compressing it.

Required cleanup stage:
- classify;
- sort;
- name;
- lane;
- boundary;
- neighbor-check;
- proof-check;
- accept / refine / park / reject.

Working form:
`Slop together -> see the shape -> clean the shape -> fit-test -> save/prove only if accepted.`

## 4. Large Model Fit Test

The “large model” idea means the house is not a flat checklist.

It is a layered, recursive working body:
- small router points to big house;
- house contains rooms;
- rooms contain tools;
- tools create outputs;
- outputs return through proof;
- proof decides whether outputs become stable;
- stable pieces can become future tools, rooms, or maps.

Candidate creation fits the large model only if it strengthens that recursion.

A new room/tool is allowed when it:
1. solves a repeated or visible work need;
2. has a clear job;
3. has a boundary;
4. supports the existing house instead of replacing it;
5. has a cleanup/proof path;
6. can return a clean output to the user;
7. does not turn chat into the house;
8. can be retired if stale.
