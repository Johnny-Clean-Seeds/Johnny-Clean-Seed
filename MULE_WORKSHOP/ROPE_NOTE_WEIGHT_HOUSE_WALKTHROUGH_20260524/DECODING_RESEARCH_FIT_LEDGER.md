# Decoding Research Fit Ledger

Status: RESEARCH WASH / FIT LEDGER / SUPPORT ONLY
Date: 2026-05-24

## Purpose

Use outside decoding ideas only when they help the house decode source, fog, proof, and note visibility without importing a full outside framework as a new boss.

## Sources Checked

| Source idea | Reference | Clean use |
|---|---|---|
| Sensemaking / data-frame loop | Klein, Phillips, Rall, Peluso, "A Data-Frame Theory of Sensemaking" | A note matters when it helps test the current frame against real evidence, or repairs a bad frame. |
| Abductive reasoning | Stanford Encyclopedia of Philosophy, "Abduction" | A note can propose the best current explanation, but it stays provisional until proof catches up. |
| Bayesian evidence updating | Bayesian updating / Bayes factor references | Weight should move with evidence and counterevidence, not volume or mood. |
| Belief/plausibility interval | Dempster-Shafer evidence theory references | When proof is incomplete, hold a bounded uncertainty range instead of fake certainty. |
| Information scent | Pirolli and Card information foraging references | Make useful notes findable by source, label, room, and trigger; do not make them loud through clutter. |

## Adopted

1. Frame repair:

Use notes to test whether the current house frame still explains the live evidence.

2. Provisional explanation:

Treat note meaning as candidate explanation until proof supports it.

3. Evidence update:

Raise or lower note visibility when evidence, counterevidence, or repeated live failure changes.

4. Uncertainty range:

Use PARTIAL, PASS WITH GUARDRAILS, UNKNOWN, or BLOCKED when proof is not enough for clean PASS.

5. Information scent:

Design notes with source, room, trigger, quiet-down condition, and return point so the right note is findable at the right moment.

## Rejected As Active

Rejected:

- importing a full external framework when a small house-native checkpoint works;
- using math-like scores as fake proof;
- letting "decoding" become mystical pattern matching;
- treating information scent as marketing polish instead of retrieval control;
- making old notes louder because they are old.

## Clean Result

Outside decoding ideas support one small local rule:

`note visibility follows evidence, fog, route need, recurrence, time-risk, and neighbor fit, then quiets when its job closes.`
