# Wrap Stage Agent Frame Ratio

Status: SYNTHESIS / SUPPORT ONLY
Date: 2026-05-24

## Core Ratio

Clean wraps need both frame and freedom.

Frame without freedom produces rigid ritual.

Freedom without frame produces smooth drift.

The working ratio is:

`enough frame to route, enough freedom to notice, enough washer to prove`

## Stage Ratios

| Stage | Frame | Freedom | Stage training |
|---|---:|---:|---|
| Source intake | 80 | 20 | Keep the original safe and named. |
| First clean translation | 70 | 30 | Convert rough pressure without losing source. |
| House walk / diagnosis | 60 | 40 | Walk rooms, notice odd pressure, keep object in hand. |
| Creative room | 40 | 60 | Let missing shapes appear, but require exit labels. |
| Washer / replay | 75 | 25 | Re-run the same issue until output changes. |
| Final wrap / save | 85 | 15 | Prove, place, receipt, commit, push, and close. |
| Signal stand-down | 70 | 30 | Decide what can sleep, what returns, and what remains blocked. |

## Adjustments By Agent

For weak/fast/instant agents:

- increase frame during intake, translation, washer, and closeout;
- require active object, source, route, and proof fields;
- use short outputs only after a visible walk has already happened;
- replay the same issue after every correction.

For strong/slower agents:

- preserve frame at boundaries;
- allow more freedom in diagnosis and creative room;
- require proof at closeout;
- avoid over-prompting the agent into rigid ritual.

For rigid agents:

- insert a signal-before-object-type pass;
- ask what is missing or overloaded;
- test one controlled route variation;
- do not drop proof gates.

For drifting agents:

- raise frame immediately;
- return to active object;
- replay the same issue through the corrected route.

## First Normal Clean Seed Wrap Implication

The first normal wraps should not be giant doctrine dumps.

They should be staged training wraps:

1. Source custody wrap.
2. Object identity wrap.
3. Room route wrap.
4. Issue replay washer wrap.
5. Creative exit-label wrap.
6. Final proof/receipt wrap.
7. Signal completion / controlled relaxation wrap.

Each wrap teaches one behavior and then tests whether the output changed.

## Ratio Guard

Do not install the ratio as a rigid numeric law.

Use it as a phase-weighting card. The real proof is whether the work becomes cleaner, more capable,
and less dependent on the user beating the same correction into it eight times.
