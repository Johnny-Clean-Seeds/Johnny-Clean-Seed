# GATE NEIGHBOR FIT MAP

Status: NEIGHBOR FIT / REVIEW ONLY / NOT PASS  
Purpose: show how the gate bloat routes without becoming dirty sprawl.

## Fit Principle

A gate is not useful by itself. It needs:

- an upstream key;
- a lawful downstream neighbor;
- proof/status surface;
- return-home state;
- blocker route if it cannot close.
