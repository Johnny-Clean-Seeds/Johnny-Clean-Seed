Status: FINAL WASH / SUPPORT ONLY

## Washer Habit

Use this as the house habit for broad, foggy, or creative work:

```text
walk
-> list
-> burn deadweight
-> wear top 1-3 candidates
-> walk again
-> creative/outside challenge
-> synthesize third shape
-> wash gates
-> return to work or lock
```

## Wash Gates

| Gate | Check |
|---|---|
| Source | Is the raw source, path, hash, or how-found proof preserved? |
| Object | What exact object is active? |
| Room | Which room owns it? |
| Relation | Is it source, checkpoint, proof, neighbor, parking, blocker, promotion candidate, receipt, or custody? |
| Authority | Is this support, rule, active guide, TODO, receipt, or local-only evidence? |
| Proof | What proves the claim and what does not? |
| Priority | Is this the next boss by safety, user command, custody risk, delay, impact, confidence, effort, and WIP? |
| Schedule | What event or cadence brings it back? |
| Burn | Has deadweight been converted into a final destination? |
| Cut | If active weight was released, is there a runner trace to root? |
| Git/local | Are root drop zone, Git status, missing tracked files, and push state checked? |
| Close | Did the result get saved with a receipt? |
